Saint-Folquin ( in flaminke: Sint-Volkwin )  ch'est ène  conmeune picarte, plachée dins ch' départémint   dech Pas-d'Caleus pi dins  l'  région Heuts-d'Franche.


== Jografie ==
Ech fleufe l'Abbe i passe ichi. Il y o étou l' rivière Mardick.


== Aménistrachon ==


== Nombe ed gins ==


== Chés monumints pi lius d'mémoére ==
L'église saint-Folquin
Ch' meunumint à chés morts
		
		


== Lius, agés, hamiaus pi écarts ==
Chl' hamiau Monnikenbure ( Mannequebeurre ) est situé pré du ru Mardick.
Chl' hamiau Hennvin aveuc ch' canal éd Calés.
